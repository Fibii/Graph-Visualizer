
## Graph Visualizer

Something i needed while studying graph theory/graph algorithms.

#### Try it
`git clone https://github.com/Fibii/Graph-Visualizer.git`

`cd Graph-Visualizer`

`npm install`

`npm start`

#### Features
- [x] Multiple Graph Layouts
- [x] Takes input from the user
- [x] Directed/Undirected Edges
